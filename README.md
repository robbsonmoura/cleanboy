# CleanBoy

Loja de roupa masculina minimalista — projeto desenvolvido para a disciplina CC2TA.

## Descrição

Projeto em React (Vite) que simula uma loja de roupas masculinas com foco em design minimalista. Inclui navegação SPA com rotas, componentes reutilizáveis, uso de props e CSS Modules.

## Páginas

* Home
* Products
* About
* Contact

## Requisitos atendidos

* React (Vite)
* Props em ProductCard
* SPA com rotas (react-router-dom)
* HTML, CSS, JavaScript
* Navbar e Footer como componentes fixos
* Responsivo
* Organização de pastas: `src/components`, `src/pages`, `src/assets`
* CSS Modules

## Como rodar localmente

```bash
git clone https://github.com/SEU_USUARIO/cleanboy.git
cd cleanboy
npm install
npm run start
# abrir http://localhost:5173
```
